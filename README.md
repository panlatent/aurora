## Seven PHP Framework

Seven 是一个Web应用开发框架, 它基于全新的PHP7设计使它可以使用所有PHP特性, 并被设计支持未来的JIT技术. Seven最大的特点是支持框架以常驻内存
的方式进行服务, 它自带了一个使用PHP编写的Http服务器.
